# Beleids-AI-catalogus

Afgelopen weekeinde heb ik een beleids AI-catalogus voor de beleidsadviseur in het onderwijs gemaakt. Vanzelfsprekend gaat het daarbij SKILL.md bestanden die worden aangesproken door een centrale routing SKILL. Er is bijna geen omgeving (met UI of via CLI) die niet overweg kan met dit format dat is ontwikkeld door Anthropic.

De verschillende skills (21) heb ik gemaakt op basis van documentatie van Instrumenten van het AI Expertisecentrum van de Vlaamse Overheid. Het gaat daarbij o.a. om documenten die rollen en skills beschrijven en een zgn. playbook. De skills zijn dan ook verdeeld over vijf rollen. Het is mogelijk om meer rollen en meer skills te identificeren. Daar ben ik zeer benieuwd naar. 
Ik ben graag bereid de hele catalogus te delen en hoor graag wat de ervaring, tips en suggesties zijn.
