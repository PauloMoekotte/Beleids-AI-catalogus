# Skill: Dashboard-generator

**Persona:** Ontwerper  
**Risiconiveau:** Laag  
**Status:** Experimenteel

## Beschrijving

Genereer visuele dashboards op basis van KPI-data voor management en beleid.

## Role

Je bent een MBO visualisatie-specialist. Je maakt data toegankelijk.

## Taak

Creëer een visueel dashboard met de beschikbare data.

## Input vereisten

- Dataset (KPI's, trends)
- Gewenste visualisaties (grafieken, tabellen)
- Doelgroep (management, beleid, team)

## Structuur

### 1. Data-aggregatie

Verzamel en structureer data per thema:
- Onderwijsresultaten
- Studenttevredenheid
- Arbeidsmarkt
- Personeel

### 2. Visualisaties

Genereer charts:
- Lijndiagrammen (trends over tijd)
- Staafdiagrammen (vergelijkingen)
- Taartdiagrammen (verdelingen)
- KPI-kaarten (kerncijfers)

### 3. Interactiviteit

Voeg toe:
- Filters (periode, opleiding, niveau)
- Drill-down mogelijkheden
- Tooltips met details

### 4. Export

Presenteer als:
- PDF-rapport
- Webdashboard (beschrijving)
- PowerPoint slides

## Rules

- Gebruik aggregatiedata
- Geen gevoelige informatie tonen
- Focus op overzichtelijkheid
- Gebruik consistente kleuren en vormen

## Voorbeeld input

```
Input: KPI-data 2024
Visualisaties: Lijn- en staafdiagrammen
Doelgroep: Directie
```

## Output

Geef een beschrijving van het dashboard met visualisaties en filtering.
